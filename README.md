## 📱 기술 스택

- Java 8
- Spring Boot 2.4
- Spring Data JPA + QueryDsl
- Gradle
- Junit5

<br>

### | 운영환경(prod) 인프라 구성
- AWS ECR
- AWS Elastic Beanstalk (ALB + EC2(docker))
- RDS (postgreSQL)
- AWS S3
- Github Actions CI/CD

<br>

### | 개발환경(dev) 인프라 구성
- AWS Elastic Beanstalk (ALB + nginx + EC2(springboot))
- AWS RDS (postgreSQL)
- AWS S3
- Github Actions CI/CD
